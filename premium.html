Premium commands

Commands unlocked with Premium

These commands are only available once your server has Premium activated. All require Manage Server permission.

+setpersonality ⭐

Give Onyx a custom personality that only exists in your server. Write anything — serious, unhinged, themed to your community.

+setpersonality <your personality text> · Manage Server
+clearpersonality ⭐

Reset Onyx's personality back to the global default. Removes your server's custom personality text entirely.

+clearpersonality · Manage Server
+viewpersonality ⭐

See the current custom personality text set for your server, or check if your server has Premium active.

+viewpersonality · Manage Server
+setserverpfp ⭐

Request a custom profile picture for Onyx in your server. Provide an image URL — the request is sent to the bot admin for approval.

+setserverpfp <image URL> · /setserverpfp · Manage Server
+clearserverpfp ⭐

Remove your server's custom Onyx PFP and revert back to the global profile picture.

+clearserverpfp · /clearserverpfp · Manage Server
+engage ⭐

Enable Onyx to randomly and naturally jump into conversations in your server without being @mentioned or called by name.

+engage · /engage · Manage Server
+disengage ⭐

Make Onyx go quiet — it will only respond when @mentioned or called by name. Turns off the random chime-ins.

+disengage · /disengage · Manage Server
+activate-ai ⭐

Make Onyx auto-reply to every single message in a channel. Full AI mode — no need to ping or call the bot.

+activate-ai · /activate-ai · Manage Server
+stop-ai ⭐

Stop Onyx from auto-replying in an activated AI channel. Returns the channel to normal mention-only mode.

+stop-ai · /stop-ai · Manage Server
+disable-ai ⭐

Block all AI chat and image generation in a specific channel entirely. Useful for keeping certain channels clean.

+disable-ai · /disable-ai · Manage Server
+enable-ai ⭐

Re-enable AI chat and image generation in a channel that was previously blocked with +disable-ai.

+enable-ai · /enable-ai · Manage Server
